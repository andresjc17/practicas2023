MEDIA QUERY

Este es un modulo CSS3 que permite adaptar la representacion del contenido a caracteristicas del dispositivo como la resolucion de pantalla o presencia de caracteristicas de accesibilidad como el braille. Las media queries (en español "consultas de medios") son útiles cuando deseas modificar tu página web o aplicación en función del tipo de dispositivo (como una impresora o una pantalla) o de características y parámetros específicos (como la resolución de la pantalla o el ancho del viewport del navegador).
